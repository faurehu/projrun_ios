##### Technologies:
* React Native -
* Redux
* Jest

##### Instructions:

Clone the repo and call:

```shell
npm install
```

Make sure the tests run by executing jest

```shell
npm test
```

Start the packager and simulator by calling the following line:

```shell
react-native run-ios
```
